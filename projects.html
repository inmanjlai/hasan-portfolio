  • Network Infrastructure Lab

    VPN, Firewall, and Remote Access Setup

    Designed and implemented a comprehensive home network lab utilizing FortiGate 100F firewall and Ubiquiti EdgeSwitch hardware. Configured both site-to-site and remote-access VPNs to enable secure connectivity between multiple locations and users. Set up internal DNS and DHCP services for efficient network management, along with port forwarding and VLAN segmentation to isolate and protect network traffic. Deployed self-hosted web services and established secure remote access for various devices, effectively simulating enterprise-grade network environments and workflows.
